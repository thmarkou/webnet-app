# Excel Import Template - Μαζική Εισαγωγή Επαγγελματιών

## Περιγραφή
Η λειτουργία "Εισαγωγή Επαγγελματιών" επιτρέπει τη μαζική εισαγωγή επαγγελματιών από Excel file (.xlsx).

## Excel Template Format

### Υποχρεωτικά Πεδία:
- **Όνομα** ή **Name** ή **FirstName** - Όνομα του επαγγελματία
- **Επώνυμο** ή **LastName** ή **Surname** - Επώνυμο (ή Full Name για όνοματεπώνυμο)
- **Email** ή **E-mail** - Email διεύθυνση
- **Επάγγελμα** ή **Profession** ή **Category** - Το επάγγελμα
- **Πόλη** ή **City** - Η πόλη

### Προαιρετικά Πεδία:
- **Τηλέφωνο** ή **Phone** ή **Mobile** - Τηλέφωνο επικοινωνίας
- **Οδός** ή **Street** ή **Address** - Οδός
- **Αριθμός** ή **Number** ή **No** - Αριθμός οδού
- **Ταχυδρομικός Κώδικας** ή **Postal** ή **Zip** - Ταχυδρομικός κώδικας
- **Περιοχή** ή **Area** ή **Region** - Περιοχή
- **Χώρα** ή **Country** - Χώρα (default: Ελλάδα)
- **Επιχείρηση** ή **Business** ή **Company** - Όνομα επιχείρησης
- **Υπηρεσία** ή **Service** - Όνομα υπηρεσίας
- **Διάρκεια** ή **Duration** - Διάρκεια σε λεπτά (default: 60)
- **Τιμή** ή **Price** - Τιμή υπηρεσίας
- **Περιγραφή** ή **Description** - Περιγραφή

## Σημειώσεις

1. **Flexible Column Names**: Το σύστημα αναγνωρίζει αυτόματα τα columns με διάφορους τρόπους (π.χ. "Όνομα", "Name", "FirstName" όλα λειτουργούν)

2. **Geocoding**: Αν παρέχονται οδός και πόλη, το σύστημα θα κάνει αυτόματο geocoding για να βρει τις συντεταγμένες

3. **City Mapping**: Το σύστημα προσπαθεί να αντιστοιχίσει την πόλη με τις υπάρχουσες πόλεις στη βάση. Αν δεν βρει, δημιουργεί νέο city ID

4. **Error Handling**: Αν κάποια rows έχουν λάθη, θα εμφανιστούν λεπτομερή error messages με το row number

## Χρήση

1. Πάτα "Εισαγωγή Επαγγελματιών" στο Database Management
2. Επίλεξε Excel file (.xlsx)
3. Περίμενε την επεξεργασία
4. Δες τα αποτελέσματα (πόσοι εισήχθησαν επιτυχώς, πόσοι απέτυχαν)
